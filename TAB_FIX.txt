================================================================================
  üêõ TROUBLESHOOTING
================================================================================

PROBLEM: Tabs still not clicking
---------------------------------
1. Check console for JavaScript errors
2. Look for red error messages
3. If you see errors, refresh page (F5)

PROBLEM: Console shows clicking but content doesn't change
-----------------------------------------------------------
This means onClick works but rendering doesn't.

Check:
1. Do you see: [AdminDashboard] Active tab changed to: [tabname]?
   - YES ‚Üí State is updating, might be CSS issue
   - NO ‚Üí State not updating, clear cache

Fix:
localStorage.clear()
Refresh page (F5)
Login again

PROBLEM: Tabs click but show "Loading dashboard..."
----------------------------------------------------
Data still loading from backend.

Check:
1. Is backend running on port 8000?
2. Console should show: [AdminDashboard] Data fetched successfully
3. If not, check backend terminal for errors

PROBLEM: See error "Failed to fetch"
-------------------------------------
Backend not responding.

Fix:
cd backend
python -m uvicorn main:app --reload --port 8000

PROBLEM: Nothing happens when clicking
---------------------------------------
JavaScript might be blocked or page not fully loaded.

Try:
1. Hard refresh: Ctrl+Shift+R
2. Clear cache: Ctrl+Shift+Delete
3. Close and reopen browser
4. Restart frontend server
