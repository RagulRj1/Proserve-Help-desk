================================================================================
  üìß STEP 1: Setup Email Notifications (5 Minutes)
================================================================================

1. CREATE GMAIL APP PASSWORD:
   
   a) Go to: https://myaccount.google.com/security
   b) Enable "2-Step Verification" (if not already enabled)
   c) Go to: https://myaccount.google.com/apppasswords
   d) Select "Mail" and "Windows Computer"
   e) Click "Generate"
   f) COPY the 16-character password (example: abcd efgh ijkl mnop)

2. CREATE .env FILE:
   
   a) Open terminal
   b) cd backend
   c) copy .env.example .env

3. EDIT .env FILE:
   
   Open backend/.env in notepad and add:
   
   ENABLE_EMAIL=true
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=your-email@gmail.com
   SMTP_PASSWORD=abcdefghijklmnop
   FROM_EMAIL=noreply@yourcompany.com
   FROM_NAME=IT Help Desk System

   ‚ö†Ô∏è IMPORTANT: Replace with YOUR Gmail and App Password!

4. INSTALL DEPENDENCIES:
   
   cd backend
   pip install python-dotenv

5. RESTART BACKEND:
   
   Stop backend (Ctrl+C)
   cd ..
   START_BACKEND.bat

‚úÖ DONE! Emails will now be sent when tickets are assigned!

================================================================================
  üì± STEP 2: Setup SMS Notifications (Optional - 10 Minutes)
================================================================================

1. SIGN UP FOR TWILIO:
   
   a) Go to: https://www.twilio.com/try-twilio
   b) Create free account (get $15 credit)
   c) Verify your phone number
   d) Get a Twilio phone number

2. GET CREDENTIALS:
   
   From Twilio Console Dashboard:
   - Account SID (example: ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx)
   - Auth Token (example: your_auth_token_here)
   - Twilio Phone Number (example: +1234567890)

3. UPDATE .env FILE:
   
   Open backend/.env and add:
   
   ENABLE_SMS=true
   TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   TWILIO_AUTH_TOKEN=your_auth_token_here
   TWILIO_PHONE_NUMBER=+1234567890

4. INSTALL TWILIO:
   
   cd backend
   pip install twilio

5. RESTART BACKEND:
   
   Stop backend (Ctrl+C)
   cd ..
   START_BACKEND.bat

‚úÖ DONE! SMS notifications will now be sent!
